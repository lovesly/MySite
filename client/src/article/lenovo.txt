美帝良心， 一句戏称， 几年来只是当做玩笑听听， 直到真的踩着了， 才恍然大悟， 原来传说中的狗屎是真的。 回头一看， 联想股价跌去 60%， 即使以金融白痴的角度看， 好像也不是好事。 17年的财报显示中国区以四分之一的销售量， 贡献了几乎全部利润。 联想 sucks 名副其实不是吗。 宏观的不谈了， 也谈不了。 就以一个消费者的角度， 说说自己的体验。 

我这个人一向运气不好，经常买到残次品，这一次又踩坑了。联想不愧是美帝良心，只要国籍不变， 上哪儿买都不省心。
回国前， 想到老爸老妈还在用我大学时用的那台老Thinkpad， 于是果断挑了一台性能爆炸的高配T480， 虽然我十分肯定他们连十分之一的性能都用不到。 
两周以后到手的这台 T480, 真是金玉其外，败絮其中。用了一天不到， 总能听到微小的爆音声， 搞得我以为自己耳机坏了， 换了耳机换了音箱依然如此。 我研究了2天， 发现外接显示器的音响没有问题， 外接显示器的音频走的是 HDMI， usb 外接 DAC 也没问题， 这是绕过了内置声卡。 最有意思的是， 把充电器拔了， 全都没问题。 搞了半天是电流干扰到声卡信号。 因为马上要回国， 也没时间退货什么的，心里很难受， 全球联保好像还给取消了。
刚刚平复了一下心情， 用了两天以后， 又发现一个问题， 有时候没干什么， 开个网页风扇也呼呼转， 老子明明在看书搞得跟在打游戏一样，资源管理器看一下， 5% 的 CPU 使用率， CPU frequency 一直保持在3.8， 3.9 GHz, i78650u 超到头也就 4.0 GHz， 联想到之前的问题， 拔掉电源， 果然， 又没事了。 T480 真的很不喜欢被插电源， 可能联想设计它的时候是打算用核动力来着。 我研究了一下电源管理， 并无卵用。 后来搜了半天， 在联想官网讨论区找到了一个解决方案。 这里是链接：
https://forums.lenovo.com/t5/ThinkPad-X-Series-Laptops/X1-Yoga-3rd-gen-CPU-frequency-pegged-at-max/m-p/4038219#M84088

打不开链接的， 请打开cmd：

    powercfg.exe /setACvalueindex OVERLAY_SCHEME_MAX SUB_PROCESSOR PERFEPP 0

    powercfg.exe /setACvalueindex SCHEME_BALANCED SUB_PROCESSOR PERFEPP 33
    
    powercfg.exe /setACvalueindex OVERLAY_SCHEME_MIN SUB_PROCESSOR PERFEPP 60

原因大概是， 一个兄弟发现联想预装的 windows 有这样一个设定， 接上电源的 CPU 即使在空闲的时候， 也要保持 max frequency （卧槽这么傲娇的吗）。 据我观察， 不少机型都有一样的设定。 我也不是搞硬件的， 实在理解不了为什么联想的 CPU 在空闲的时候也要发热。 真是猜不透联想啊。 我预感以后还会遇到各种其他的混账问题， 好在这是给爸妈买的， 他们只用来看看视频上上网。 thinkpad 看看就行，给其他品牌一点机会。 最后， 再买联想， 我是狗。

update: 
https://support.lenovo.com/dk/en/solutions/ht506625

官方补丁来了， 估计是全新的电源方案， 调整了 CPU 在连接电源情况下的频率和风扇阈值。 

我突然想起来这其实是第二次被坑。 刚入学的时候入手一台联想 Y 系列， 数据是很好看了， 就跟介绍相亲一样， 总有的夸不是， 好看的说好看， 不好看说温柔， 不温柔说善良， 能干的说能干， 不能干的说踏实， 总之就是这一套了， 中国消费者耳熟能详。 头三个月简直和带婴儿一样， 天天搞事， 经常蓝屏， “不好意思， windows 遇到了一个问题， 正在尝试解决。。。” 然后死机。 三个月我逛遍各大论坛， 数次拆机， 换固态， 做系统迁移， 蓝屏问题依旧存在， 但是重启快多了， 瞧瞧哥们儿这脑子， 啧啧。 最后官方出了一些更新， 手动安装一些特定版本的驱动， 稀里糊涂的解决问题， 有点儿像祖传代码，注释写着“我不知道为什么行， 我只知道改了就不行， 别碰， 谢谢”。 好了伤疤忘了疼， 4年过去我把这茬儿忘了。 

前几天突然蹦出来一个联想的新闻， 什么联想不是中国公司，一堆人喷联想， 过几天 CEO 赶紧出来澄清， 哎呀我从来没说过这话， 这是外媒曲解。 为什么人们会轻易相信呢， 因为在普通人看来， 联想确实不像一个中国公司。 去年 5g 投票， 联想两票全给了高通， 短码标准也落到高通手里。 之前的黑历史又特别多， 这叫什么？ 这叫欠黑。 可能是我孤陋寡闻， 我至今没见过什么产品是在本国生产， 全线产品售价均高于美国， 高配的 X1 甚至能贵出1万块。你说这年头，5毛是真不好当， 微博骂公知， youtube 喷台独， 贸易战打响， 这边刚表态， 你美帝欺人太甚， 我要支持国货。 联想跳出来， 买我的吧， 只比美国售价贵1万。 你把自己当有志青年， 人家把你当爱国韭菜。 好在现在韭菜智力渐长， 以后就没有那么好收割了。
